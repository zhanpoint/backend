## 代码生成规则

- 模块化设计
- 命名规范

- 包含简洁清晰的注释，代码文档化
- 代码复用性高，易于维护管理
- 配置与代码分离

### 流程应该是：

```
前端                              后端(Django)                      外部服务
  |                                  |                                |
  |--- 发送手机号 ------------------>|                                |
  |                                  |--- 验证手机号格式              |
  |                                  |--- 生成随机验证码              |
  |                                  |----------------------> 调用SMS API
  |                                  |                                |
  |                                  |--- 将验证码存入Redis(设置60s过期)|
  |<-- 返回发送成功（不返回验证码）--|                                |
  |                                  |                                |
  |--- 用户输入验证码并提交 -------->|                                |
  |                                  |--- 从Redis获取验证码并比对     |
  |<-- 返回验证结果 -----------------|                                |
```
